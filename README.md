## âš™ï¸ Data & Configuration

### Import / Export Settings
You can easily save and transfer your entire setup. In the **Timer Settings** panel, you will find **Import** and **Export** buttons.
- **Export:** Click this to download a `solomon-christ-ai-app-settings.json` file containing all your timer durations, custom audio URLs, tasks, and stats.
- **Import:** Click this to upload a previously exported `.json` file to instantly restore your configuration on any device.

### Hard-Coding Default Values (For Developers)
If you want to create a custom, offline version of the app with your own default settings, you can edit the configuration files directly:
- **Timer Durations & Sounds:** Open `constants.ts` and modify the values inside the `DEFAULT_TIMER_SETTINGS` object.
- **Music & Themes:** In the same file (`constants.ts`), you can change the default MP3 URLs for `FOCUS_MODES_CONFIG` and `THEME_CONFIG`.
